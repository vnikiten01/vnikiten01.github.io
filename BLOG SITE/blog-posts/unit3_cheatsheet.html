Why is JavaScript popular in web development? Let us start with the fact that every browser supports it. Whether you are using Internet Explorer, Chrome or Firefox. The next big reason is JavaScript is fast; thanks to being supported by all web browsers. JavaScript code is added directly to a HTML page. This bypasses the need for the browser to upload virtual machines, servers or compiling programs remotely. In the past the browser would have to send messages between the servers and browser slowing down the process of loading up pages or displaying images. Now much of this takes place on the side of the client (person accessing the web pages) and the servers are free to handle more important tasks. To understand this concept better, you are using a webpage where it asks for credit card information. A JavaScript page will check if the credit card number appears valid (does it have 16 digits) before sending the information to the web servers for further validation.
