My Reflection

Describe responsive design summary, the spectrum of screen sizes and resolutions is widening every day, and creating a different version of a website that targets each individual device is not a practical way forward. This is the problem that responsive web design addresses head on.

Responsive websites are built on a fluid grid and use media queries to control the design and its content as it scales down or up with the browser or device. The overall goal of responsive design is planning for the furture when unknown browsers and resolution sizes are implements.
